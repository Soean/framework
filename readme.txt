== Description ==

Assely is a PHP framework which brings a little joy to the WordPress development. Develop structured, easily scalable and complex websites or web applications with expressive and elegant syntax.

Today, WordPress development is pretty rough and messy. The Assely framework aims to introduce some standarized and comfortable ways for creating WordPress powered applications. We still have a long way to go, but we believe that it is possible.

### Interested? Take a look at documentation.
* [Project website](http://assely.org)
* [Documentation](http://assely.org/docs)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/assely-framework` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Follow up [documentation](http://assely.org/docs)
